# 合成大西瓜

奶奶意外的会玩这种游戏,网上的和小程序里面的全是广告,于是在网上找了这个 HullQin 大佬的开源版本,用AI改了一下弄出来一个没广告的给俺奶奶玩

## 功能特性

- 🎮 经典合成大西瓜玩法
- 💾 支持保存、加载游戏进度
- 📱 PWA 支持，可安装到手机(自己用EDGE浏览器测了一下开启添加快捷方式权限才可以)
- 🎨 精美水果动画效果(哈哈明明是抄的)
- 🚀 基于 Pixi.js + Box2D 物理引擎

## 在线体验

访问 [GitHub Pages](https://zyh8798.github.io/madeWatermelon/) 地址即可在线体验

## 安装到手机

1. 用Edge 或者 chrome 访问游戏地址
2. 点击"添加到主屏幕"
3. 即可像原生 APP 一样使用
